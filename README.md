# tick-tack

Play tic-tac-toe in your terminal

## Build

    $ lein ubjerjar

## Usage

Just run

    $ java -jar tick-tack-0.1.0-standalone.jar

## Playing

Use the arrow keys to move, and `Enter` to place your mark.
